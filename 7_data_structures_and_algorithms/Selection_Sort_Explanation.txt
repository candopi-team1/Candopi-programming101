(1) Find the smallest element, place it in the current position.
At the beginning, the current position is 0.
So the smallest element becomes the first element.
You array now looks like = [smallest, .....]

(2) So, in the next round of the outer loop, you begin with the 2nd position.
The inner loop finds the next smallest and places it in 2nd position.
You array now looks like = [smallest, 2nd-smallest, .....]


(3) It goes on till the end.
You array now looks like = [smallest, 2nd-smallest, ....., the least-smallest or the biggest]
