Object Orientation Lesson

Animal is abstract. It's a super/parent or top-most class. Other classes descend from it.

Other classes are concrete. They are sub classes or children classes.

All subclasses recieve the properties and functionalities of the parent. They get automatically:

  1. all properties declared in the super-class
  2. all functionalities defined in the super-class

If a subclass doesn't rewrite a function, it uses the super-class's function. E.g.In our code, only the Human class re-writes all the 3 functions : move(),fight() and eat().

A subclass can have more properties and functions, in addition to those inherited from the parent/super-class. e.g. A shrak has jaw-stregth property. A human has an education property. e.g. Only human can dance in the code.
